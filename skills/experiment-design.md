# Skill: Experiment Design

## Use When

A product change needs validation through an experiment or measured rollout.

## Workflow

1. Define the hypothesis.
2. Identify the target segment.
3. Define control and treatment.
4. Select primary metric and guardrails.
5. Estimate duration and sample needs.
6. Define ship, iterate, and stop criteria.
7. Plan analysis and monitoring.

## Output

An experiment spec with a clear decision rule.

## Quality Check

The experiment should be capable of changing the team's mind.
