# Skill: UX Review

## Use When

A flow, design, or implemented interface needs critique.

## Workflow

1. Identify the user goal.
2. Walk through the main flow.
3. Check hierarchy, labels, affordances, and feedback.
4. Review loading, empty, error, success, and permission states.
5. Check accessibility and responsive behavior.
6. Identify the highest-impact friction.
7. Recommend concrete improvements.

## Output

A prioritized UX review with specific issues and suggested changes.

## Quality Check

The review should improve clarity, confidence, and task completion.
