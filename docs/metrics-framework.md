# Metrics Framework

Use this to define how the project measures product impact.

## Metric Types

### North Star Metric

The durable measure of product value.

### Input Metrics

Leading indicators the team can influence directly.

### Output Metrics

Business or user outcomes expected to move over time.

### Guardrail Metrics

Measures that should not degrade while optimizing the primary metric.
